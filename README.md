# MACHINE-LEARNING-MODEL-IMPLEMENTATION

## Objective

Build a predictive machine learning model using Scikit-Learn for spam email detection.

## Features

- Data Preprocessing
- TF-IDF Vectorization
- Naive Bayes Classification
- Model Evaluation
- Custom Prediction

## Technologies Used

- Python
- Scikit-Learn
- Pandas
- NumPy
- Jupyter Notebook

## Installation

```bash
pip install -r requirements.txt
```

## Run

```bash
jupyter notebook
```

Open:

ML_Model_Implementation.ipynb

## Output

- Accuracy Score
- Classification Report
- Confusion Matrix
- Spam Prediction
